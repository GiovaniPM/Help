## Dir structure
>```
>~/
>|   .bash_aliases
>|   .bash_history
>|   .bash_profile
>|   .bashrc
>|   .gitconfig
>|   .gitignore
>|   download.sh
>|   upload.sh
>|   ...
>├───<projects dir>/
>|   |   ...
>|   ├───<repository>/
>|   |   |   setup.sh
>|   |   |   ...
>|   |   └───.../
>|   └───.../
>└───.../
>```

## Run download
>To run:
>``` bash
>download <repository>
>```

## Run upload
>To run:
>``` bash
>upload
>```

## Setup
>Get the file setup.sh on this repository and copy to your projects dir
>
>To run:
>``` bash
>./setup.sh
>```
